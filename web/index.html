MiMo 会话 (Cookies)

MiMo 会话是通过小米账号兑换获得的 serviceToken + xiaomichatbot_ph，用于实际 API 调用。
输入小米账号的邮箱/手机号及密码。如果账号启用了两步验证，请在点击登录后，根据提示完成验证。
使用已有的小米 passToken 和 userId 直接兑换 MiMo 权限，无需输入密码，也不会触发二次验证。

如何获取 cURL 命令？
  1. 在浏览器中登录 aistudio.xiaomimimo.com
  2. 按下 F12 打开开发者工具，切换到 Network (网络) 标签
  3. 在网页中发送一条对话，在列表中找到名为 chat 的请求
  4. 右键点击它 → Copy (复制) → Copy as cURL (bash)
  5. 将复制的内容粘贴到下方文本框中
快速导入：直接粘贴包含所有 Cookie 的完整字符串（用英文分号分隔），点击下方按钮即可自动拆分填充。
